

                             Vim Guide

                        (caps matters here!)


Recommendation

    1. Unbind the caps lock key - messes up your inputs if you accidentally
       engage it

    2. Unbind the arrow keys - do all navigation with < hjkl > in vim

-------------------------------------------------------------------------------

Starting out / Netrw (file viewer)

    Open vim in directory with Netrw        [ nvim . ]
    Go up a directory in netrw              [ ctrl - ]
    Search in Netrw                         [ /query ]
    Return to Netrw after opening file      [ space v ]

-------------------------------------------------------------------------------

Finding files

    FindFiles feature (file names)          [ space , ff ]
    LiveGrep feature (contents of files)    [ space , fg ]
    List open buffers                       [ space , fb ]
    Search help tags                        [ space , fh ]

-------------------------------------------------------------------------------

Splits

    Open same file in Horizontal split      [ ctrl w , v ]
    Open same file in Vertical split        [ ctrl w , s ]

    Open Horizontal split with Netrw        [ space , V ]
    Open Vertical split with Netrw          [ space , S ]

    Equalize split sizes                    [ ctrl w , = ]


Navigate splits (if one of the splits is Netrw, use the ctrl w version)

    Go Left                                 [ ctrl w , h ]   or   [ ctrl h ]
    Go Right                                [ ctrl w , l ]   or   [ ctrl l ]
    Go Down                                 [ ctrl w , j ]   or   [ ctrl j ]
    Go Up                                   [ ctrl w , k ]   or   [ ctrl k ]


Terminal in a split

    Start terminal                          [ :term ]
    Kill terminal                           [ :q ]

    Start typing in terminal                [ i ]
    Stop typing in terminal                 [ ctrl q ]

-------------------------------------------------------------------------------

Substitutions

    Hint:
        These are commands with specific syntax, ran in command
        mode in vim, but can be super powerful


    Global substitution                     [ :%s/<replace/<with>/g ]

                                              (replaces on all lines)


    Line range subsitution                  [ :1,5s/<replace>/<with>/g ]

                                         (replaces on lines 1 through 5)


    Relative line subsitution               [ :,+5s/<replace>/<with>/g ]

                      (replaces from current line through 5 lines after)

Plugin: markdown-preview

    Note:
        Opens a live preview of the markdown file in your browser.

    Toggle markdown preview                 [ Ctrl m ]

-------------------------------------------------------------------------------

Plugin: vim-signify

    Note:
        Shows git diff markers in the gutter. The plugin adds
        default keymaps for navigating hunks.

    Jump to next hunk                       [ ]c ]
    Jump to previous hunk                   [ [c ]

-------------------------------------------------------------------------------

Plugin: nvim-surround

    Note:
        ")" character can be replaced with others like: ], }, " etc
        For most of these, cursor has to be positioned within the thing to wrap

   Wrap inner string with "("               [ ysiw) ]
   Wrap inner string with "( "              [ ysiw( ]

   Wrap outer string with "("               [ ysiW) ]
   Wrap outer string with "( "              [ ysiW( ]

   Change quotes around string              [ cs'" ]
   from ' -> "

   Delete ")" around string                 [ ds) ]

-------------------------------------------------------------------------------

Plugin: wrapping-nvim

    Note:
        Hard wrap mode: sets 80 char limit at which a \n will be inserted
        Soft wrap mode: breaks lines at window edge with a line break symbol

    Hard wrap mode                          [ Space , wh ]
    Soft wrap mode                          [ Space , ws ]
    Toggle wrapping mode                    [ Space , wt ]
